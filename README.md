# PepWM
PepWM is written in Rust its a "battery included" tiling wayland compositor
it is designed to be fast and lightweight while also modular and customisable

# Installing
*from source*
1. install Rust, Wayland, and xorg-server
2. run `cargo build`
3. run `cargo run`
4. run `./pepwm`
